#### B2. **MSI/MMI定量分析**
**当前问题**:  
虽然提出了"氧诱导表面偏析"理论,但缺乏定量指标

**定义**:
- **MSI** (Metal-Support Interaction): Pt-O键合强度
- **MMI** (Metal-Metal Interaction): Pt-Pt键合强度

**计算方法**:
```python
# 从LAMMPS dump中提取径向分布函数 g(r)
# 1. 计算 g_Pt-O(r) 第一峰面积 → MSI_strength
# 2. 计算 g_Pt-Pt(r) 第一峰面积 → MMI_strength
# 3. 比率 MSI/MMI 随O含量变化

# 或使用键合数统计:
# MSI = <Pt-O键数> / Pt原子数
# MMI = <Pt-Pt键数> / Pt原子数
```

**关键对比**:
| 系列 | MSI/MMI | Tm (K) | 结论 |
|------|---------|--------|------|
| O0   | 0       | 1011   | 基准 |
| O1   | 0.15    | 693    | 适中稳定 |
| O2   | 0.28    | 560    | 过度氧化? |

#### B3. **热容跃变不足问题**
**当前问题**:  
`energy_analysis_v2_report.txt` 仅识别出 **1个熔点** (Cv-2, Tm=1100K)

**原因分析**:
1. **温度间隔过大** (50-100K)
   - 熔化跃变可能发生在间隔之间
2. **Cv计算方法** 可能不敏感
   - dE/dT方法需要平滑能量曲线

**补救措施**:
- 使用 **Lindemann δ** 作为主要熔点判据
- Cv跃变仅作为**辅助验证**
- 在论文中明确说明: "由于温度采样间隔限制,热容跃变识别不完整"
